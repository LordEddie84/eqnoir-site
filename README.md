# EQNOIR Website

Fully static, GitHub Pages-ready website.

## Features
- Smooth, live crypto ticker (EQNOIR, BTC, ETH, SOL)
- Hero, About, NoirHause, Whitepapers, Social Buttons, Contact, Footer
- Contact email: eqnoirproject@gmail.com
- Privacy warning included
- Dark-futuristic styling with smooth scrolling ticker

## Launch Instructions
1. Create a new GitHub repository (`EQNOIR`)  
2. Copy all files and folders (`index.html`, `style.css`, `icons/`, `whitepapers/`, `README.md`)  
3. Go to Settings → Pages → Branch: main / folder: root → Save
